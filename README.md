\# 🧬 Termux‑Doctor  

\### Cyber‑Diagnostic Suite for Termux  

Created by \*\*gh0usts3c-crypto\*\*



Termux‑Doctor is a modular, cyber‑themed diagnostic tool designed to analyze, repair, and report on the health of your Termux environment.  

It provides clean, colorized output, detailed logs, and a professional CLI experience.



\---



\## 🚀 Features



\### 🔍 System Diagnostics

\- CPU model + architecture  

\- Kernel version  

\- Memory info  

\- Uptime  

\- Termux architecture  



\### 📦 Package Diagnostics

\- Broken package detection  

\- Update availability  

\- APT database health  

\- Essential package check  



\### 🌐 Network Diagnostics

\- Local + public IP  

\- Ping test  

\- DNS resolution  

\- Interface summary  



\### 💾 Storage Diagnostics

\- Internal storage usage  

\- Home directory size  

\- External storage detection  

\- Termux storage permission check  



\### 🧬 Environment Diagnostics

\- PATH integrity  

\- Shell type  

\- Termux environment variables  

\- Common misconfiguration detection
